Терсаамский-Русский Словарь
В словаре представлены только несколько основных лексических единиц, собранные сотрудниками Kola Saami Documentation Project (DoBeS).
Русский-Терсаамский Словарь
Этот словарь является зеркальным отображением Тер-саамско-Русского словаря (смотри выше).
Tа̄рьйса̄мь-Рӯшш Соагкнэһкь
Соагкнэгкесьт вӯзьха ле̄в лышэ туаррьей са̄нҍ. Со̄нн кырьйха ля̄й проектэсьт Kola Saami Documentation Project (DoBeS).
Рӯшш-Tа̄рьйса̄мь Соагкнэһкь
Tэ̄дт соагкнэһкь лӣ та̄рьйса̄мь-рӯшш соагкнэгкь кыјјм-ка̄ррь вэ̄зхэллмужэнҍ (паййла кӣч).
